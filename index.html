Description

A system that takes care of all your splitting needs! All you need to do is:

  1. Attach the splitable script to any convex 3D game object you'd like to split.
  2. Drag the "Cutting Edge" prefab into your hierarchy.
  3. Set the layer of any gameObject you would like to avoid splitting to "Ignore Raycasts"
